🛡️ The safety spine — why a non-technical client can't break anything

Three independent guarantees run through every lane:

  • Frozen template — the AI/client never writes HTML or CSS, only fills tagged content slots. Layout damage is structurally impossible.
  • Guardian — every write (click, chat, AI) passes one deterministic validator: blocks injection, re-renders to prove all sections survive, caps blast radius.
  • Versioned + atomic deploy — bad publish? One-click rollback re-points to a prior immutable release. The live site never half-changes.
